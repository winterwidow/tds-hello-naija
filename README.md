# tds-hello-naija

A tiny Python greeter, published as part of **Tools in Data Science** at IIT Madras (May 2026).

## Install

```bash
pip install tds-hello-naija
```

## Usage

```
from tds_hello_naija import greet, shout
print(greet("TDS"))
print(shout("TDS"))
```
